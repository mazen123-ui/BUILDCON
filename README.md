# BUILDCON - Construction Landing Page

**BUILDCON** is a modern, responsive landing page for a construction company, built with **Bootstrap 5**. The project showcases a clean and professional design tailored for businesses in the construction and architecture industry.

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.

- **Interactive Sections**:
  - **About**: Company mission and quality assurance.
  - **Services**: Highlights of core business offerings (Quality, Sustainability, Integrity).
  - **Portfolio**: A showcase of completed projects with image cards.
  - **Team**: A section dedicated to the professionals behind the work.
  - **Contact**: A functional-ready contact form for client inquiries.
- **Scroll Animations**: Smooth entrance animations for sections using the `IntersectionObserver` API.
- **Scroll Improvements**: Includes a "Back to Top" button for enhanced navigation.
